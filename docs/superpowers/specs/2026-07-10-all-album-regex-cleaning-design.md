# 所有专辑统一正则清洗设计

## 目标

所有扫描出的专辑，无论是否已有封面，都先经过确定性的 `AlbumNameCleaning` 展示名称清洗。封面状态只用于封面墙筛选和后续处理，不再决定名称是否清洗。

## 边界

- 复用 `AlbumDisplayNameCleaning.displayNames` 和现有 70% 标签多数票规则。
- 无封面专辑与有封面专辑使用完全相同的目录名、标签佐证、占位值和误伤保护规则。
- 仅改变派生展示名和搜索词，不修改原始目录、音频文件、标签、扫描边界、快照或封面保存位置。
- 不借此自动触发 Ollama；已有增强建议仍优先于标签候选。

## 验证

- 将原“无封面不清洗”测试改为“无封面也清洗”，并同时覆盖歌手名、专辑名和搜索词。
- 运行名称清洗专项测试、全库审计和 `Scripts/verify.sh`。
- 全库审计应显示真实运行时与强制展示链路变化数一致，质量门禁继续通过。
